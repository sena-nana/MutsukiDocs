## 0.4.6_20221022
### 新功能
- 现在会通过AI判断生成后的图片是否nsfw，并将不同判断结果的图片存储在单独文件夹中
- 现在可以修改单群的cd
    - 在需要修改的群内使用.aidraw set cd 120将cd修改为120
    - 在群内使用.aidraw set查看本群的设置
### 修复
- 删除了启动时版本自检以解决与部分插件冲突和在Unix系统上报错的问题
- 在H屏蔽词中添加”裸“以解决中文绕过nude关键词的问题
### 更改
- 整合了以图生图和文本生图的处理逻辑
